Rejoindre une partie (lancé depuis le serveur)

Attributs :
    command : la commande décrite, ici "game_join"
	player_id : id du joueur (pour savoir quelles légions sont contrôlées)
	team_id : id de l'équipe du joueur

Exemple :
{
    "command": "game_join",
    "player_id": 0,
	"team_id": 1
}
